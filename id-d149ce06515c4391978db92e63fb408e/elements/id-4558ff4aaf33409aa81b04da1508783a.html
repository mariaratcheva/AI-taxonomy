Creative Writing ()
The generative capabilities of AI systems are based on exiting data. AI can generate creative outputs (art, music, design) by learning from existing data, but its creativity is bounded by patterns found in the data it was trained on. AI lacks the original, intuitive, and emotional depth that characterizes human creativity. Use Case: Generate an answer of an email SBBs: Generate audio– voice synthesis Generate a summary of a document Generate text, story, art, code– deepfakes, GPT, BERT, T5, CoPilot Create a video - VGAN (Video Generative Adversarial Network) Create an image - StyleGAN2 (NVIDIA), OpenAI's CLIP, DALL-E – creates images from text description Tools to fight deepfakes – in progress
